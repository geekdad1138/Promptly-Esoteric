PromptlyEsoteric 🌀
An experimental framework designed to test the limits of LLM reasoning through the lens of Esoteric Programming Languages (Esolangs).

Inspired by recent research (e.g., USC’s Idris/GPT-5 study), this repository documents the process of using closed-loop compiler feedback to "teach" AI how to program in languages with near-zero training data.

🔬 The Hypothesis
Most LLMs are trained on massive datasets of Python, JavaScript, and C++. By forcing an AI to write in languages like Malbolge, Brainfuck, or LOLCODE, we strip away its "memory" of common patterns and force it to rely purely on:

Syntactic Logic: Following a provided grammar or specification.

Iterative Correction: Learning from interpreter/compiler error logs in a recursive loop.

Abstract Reasoning: Solving logic puzzles in environments designed to be "human-impossible."

🛠️ The "Babel Loop" Workflow
This project utilizes a feedback loop between the LLM and the local environment:

Prompt: The AI is given the language specification and a task (e.g., "Output 'Hello World'").

Execution: The generated code is run through a local interpreter (via VS Code/Terminal).

Feedback: If it fails, the error log (or incorrect output) is fed back to the AI.

Refinement: The AI adjusts its logic until the unit test passes.

def test_code(file_path, expected_output="Hello World"):

    try:
        # Run the malbolge code
        actual_output = eval(code)
